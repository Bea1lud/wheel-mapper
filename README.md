# Wheel Mapper

Wheel Mapper — приложение на Python для настройки игровых рулей и привязки кнопок, педалей и осей к клавишам клавиатуры.

Проект создан для упрощения использования рулей в играх, которые не поддерживают определённые модели устройств или имеют ограниченные возможности настройки управления.

## Возможности

* Поддержка игровых рулей через Pygame
* Создание и загрузка профилей
* Привязка кнопок руля к клавишам клавиатуры
* Обработка осей поворота и педалей
* Настройка мёртвых зон (Deadzone)
* Сохранение настроек в JSON
* Удобный интерфейс на CustomTkinter
* Возможность создавать отдельные профили для разных игр

## Используемые технологии

* Python
* Pygame
* CustomTkinter
* PydirectInput
* JSON

## Установка

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Принцип работы

1. Программа обнаруживает подключённый руль.
2. Пользователь создаёт или выбирает профиль.
3. Настраивает привязки кнопок и осей.
4. Настройки сохраняются в JSON-файл.
5. Во время работы приложения действия руля преобразуются в нажатия клавиш клавиатуры.
